## The 3×3 grid map
Instagram fills the grid top-left → right, newest first, so post them in **reverse order** (08 first, 00 last) and then **pin the intro** so it holds top-left.

```
[ 00 Intro    ] [ 01 Sankofa  ] [ 02 Akwaaba   ]
[ 03 Day names] [ 04 Gye Nyame] [ 05 Medaase   ]
[ 06 Ɛte sɛn? ] [ 07 Dwennimmen] [ 08 Greetings ]
```
Mix: 1 intro · 3 Word-of-the-Day · 3 Adinkra · 2 Culture.
